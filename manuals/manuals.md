# マニュアル

# パッケージのインストール

```bash
# prettierをインストール
npm install --save-dev prettier eslint-config-prettier eslint-plugin-prettier
# socket.io-clientをインストール
npm install socket.io-client
# react-hook-formをインストール
npm install react-hook-form
# zustandをインストール
npm install zustand
# three.jsをインストール
npm install @react-three/drei @react-three/fiber three
# lucide-reactをインストール
npm install lucide-react
# next-themesをインストール
npm install next-themes @types/next-themes
# supabase-jsをインストール
npm install supabase-js @supabase/auth-helpers-nextjs
# framer-motionをインストール
npm install framer-motion
# sonnerをインストール
npm install sonner
```

## shadcn/uiのインストール

```bash
# shadcn/uiの初期化
npx shadcn@latest init
# toastコンポーネントを追加
npx shadcn@latest add toast
# buttonコンポーネントを追加
npx shadcn@latest add button
# checkboxコンポーネントを追加
npx shadcn@latest add checkbox
# dialogコンポーネントを追加
npx shadcn@latest add dialog
# inputコンポーネントを追加
npx shadcn@latest add input
# labelコンポーネントを追加
npx shadcn@latest add label
# sonnerコンポーネントを追加
npx shadcn@latest add sonner
# progressコンポーネントを追加
npx shadcn@latest add progress
```

# マニュアル

## パッケージのインストール

```bash
# prettierをインストール
npm install --save-dev prettier eslint-config-prettier eslint-plugin-prettier
# socket.io-clientをインストール
npm install socket.io-client
# react-hook-formをインストール
npm install react-hook-form
# zustandをインストール
npm install zustand
# three.jsをインストール
npm install @react-three/drei @react-three/fiber three
# lucide-reactをインストール
npm install lucide-react
# next-themesをインストール
npm install next-themes @types/next-themes
# supabase-jsをインストール
npm install supabase-js @supabase/auth-helpers-nextjs
# framer-motionをインストール
npm install framer-motion
# sonnerをインストール
npm install sonner
# React Hook Form をインストール
npm install react-hook-form
# Zod (スキーマバリデーションライブラリ)
npm install zod @hookform/resolvers
```

## テストの環境構築

### jestのインストール

```bash
# Jestのインストール
npm install --save-dev jest @types/jest
# ts-jestのインストール
npm install --save-dev ts-jest
# React Testing Libraryのインストール
npm install --save-dev @testing-library/react @testing-library/jest-dom
# その他の関連ライブラリ
npm install --save-dev @testing-library/user-event jest-environment-jsdom
```

### E2Eテストの環境構築

```bash
# Playwrightのインストール
npm install --save-dev @playwright/test
# Playwrightのブラウザをインストール
npx playwright install
# dotenvのインストール
npm install dotenv
# Playwrightのテスト結果をHTMLで出力
npx playwright show-report
```

## shadcn/uiのインストール

```bash
# shadcn/uiの初期化
npx shadcn@latest init
# toastコンポーネントを追加
npx shadcn@latest add toast
# buttonコンポーネントを追加
npx shadcn@latest add button
# checkboxコンポーネントを追加
npx shadcn@latest add checkbox

npx shadcn@latest add dialog
# inputコンポーネントを追加
npx shadcn@latest add input
# labelコンポーネントを追加
npx shadcn@latest add label
# sonnerコンポーネントを追加
npx shadcn@latest add sonner
# progressコンポーネントを追加
```
